# LeoNes — Visual Identity System

## Logo direction
Wordmark-led, not mascot-led (mascots date fast, wordmarks compound in value). "LeoNes" set in a rounded-geometric display face, the capital N doing double duty as a subtle peak/mountain-ear shape. A single mark accompanies it: **a minimalist lioness silhouette made from one continuous sparkle-trail line** — usable alone as an app icon / embossed foil stamp once the brand is established, the way the Nike swoosh outlives "Nike." No cartoon face, no googly eyes — keep it premium, not licensed-character.

Lockups:
- Primary: wordmark + mark, horizontal.
- Compact: mark alone (packaging emboss, favicon, charm shape).
- Never stretch, rotate, or recolor the mark outside the approved palette below.

## Color system
Named, not just hex — names are what get used in copy decks and Figma layers.

| Name | Hex | Role |
|---|---|---|
| Crystal White | #FFFFFF | Base canvas, negative space |
| Cloud | #F7F5F3 | Off-white background, warmer than pure white |
| Pastel Sky | #C9E4F5 | Primary blue — trust, calm, "parent-safe" cue |
| Baby Pink | #FBD3DE | Primary pink — warmth, magic |
| Lioness Gold | #F0C36B | Accent only — the "confidence" color, used sparingly (foil, sparkle, CTA hover) |
| Mane Amber | #E8A24C | Deeper gold for contrast/depth, never large fields |
| Ink Plum | #3A2E3D | Text color instead of black — softer, still readable |
| Sparkle Silver | #E7ECF0 | Dividers, shimmer overlays |

Rule: **every layout is 70% white/cloud, 20% one pastel, 10% gold accent.** If a page feels loud, remove pink before you remove white.

## Typography
- **Display / headlines:** a rounded-geometric sans with warmth (think Fraunces-adjacent for occasional editorial moments, but primary display is rounded sans — e.g. Quicksand/Nunito-class shapes, licensed equivalent to be selected: candidates **Gasoek One** for hero wordmark moments, **Poppins/Nunito** for everyday display).
- **Body:** a clean humanist sans (Inter or General Sans) — this is where parent-trust lives, so body copy must read as competent and clear, not cutesy. Never use a display/rounded font for ingredient lists or safety copy.
- **Never** use a script/cursive font as primary — it reads "craft store," not premium.

## Shape language
- Rounded corners everywhere, minimum radius scales with element size (buttons 12–16px, cards 24–32px, hero containers up to full pill/blob shapes).
- Soft blob / cloud shapes as background containers instead of hard rectangles.
- Sparkle glyph (4-point star) as the only "icon accent" — used like a punctuation mark, never more than 2–3 per screen.
- Bubbles and tiny hearts reserved for motion (see below), not static clutter — a static screen full of bubbles reads cheap.

## Photography direction
- **Lighting:** bright, soft, diffused — no hard shadows, no moody/dark beauty lighting (that's Rare Beauty/adult territory).
- **Backgrounds:** crystal water, floating clouds, soft gradient sweeps, marble-meets-cotton-candy surfaces.
- **Every hero shot must be re-shootable as a transparent PNG** — plan the lighting so product can be cleanly cut out for the website and social templates.
- Mandatory shot list per SKU: hero (transparent), lifestyle (in a girl's hand, age-appropriate, never overly styled/adult), macro texture, packaging closed, packaging open/unboxing sequence, product on water, product with suspended sparkle/glitter, before/after only where genuinely truthful (e.g., color-changing gloss swatch).

## Motion language
- Micro-interactions: soft ease, slight overshoot (like a bubble settling), never sharp/linear easing — nothing should feel "corporate SaaS."
- Floating particles (sparkles, tiny hearts, bubbles) drift slowly in page backgrounds, respond subtly to cursor/scroll, always low-opacity so they don't fight content.
- Page transitions: soft cross-fade + gentle scale, 250–400ms. No hard cuts, no slide-ins from screen edge (feels like a mobile app menu, not a boutique).

## Packaging design rules
- Packaging is glossy, weighted, satisfying to hold and reopen — this *is* the product experience as much as the formula.
- Every SKU's outer packaging must be sticker/charm-collectible on its own (kids display and trade packaging even when empty — design for that second life).
- One "reveal" moment is mandatory: a lid that changes color with warmth, a window showing suspended glitter, foil that catches light when tilted. Packaging without a reveal moment does not ship.
