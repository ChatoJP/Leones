# LeoNes — Viral Playbook: Gloss Collection 01 (Lips)

Format: every clip is 8s, 9:16, loops seamlessly. Post natively on TikTok, IG Reels, YT Shorts.
Text overlays and audio get added at post time (never baked into the video) so one clip
becomes dozens of variants by swapping hook text + sound.

## The universal formula
- **0–2s:** visual surprise already in motion (never a logo, never a fade-in).
- **2–6s:** one escalating gag. One. Not three.
- **6–8s:** punchline that loops invisibly back to frame one — loop = rewatches = the algorithm's favorite meal.
- **Caption = a question kids answer in comments.** Comments > likes for reach.
- **Audio:** always pick from the trending page THAT DAY within these categories; never reuse last week's sound.

## Posting system
1. **Launch order:** 008 (boop) → 004 (team war) → 003 (glitter freeze) → rest at 1/day.
2. Post each clip 3× over 3 weeks with different hook text + audio — same video, new joke.
3. Reply to top comments with follow-up videos ("you asked what's inside…") — comment-to-content pipeline.
4. Duet/stitch everything kids make within 24h. Speed of response is the community strategy.
5. Never say "buy." The bio link and the pinned comment ("they're real 🫢 leones.co") do the selling.

## Scaling with AI
Every concept above regenerates infinitely: swap the backdrop (bedroom, beach, locker),
the season (snow for winter drop), or pair two products in one gag (jelly boops the
sleepy jar awake). The manifest + pipeline in `tools/generate-assets.mjs` makes each
variant one prompt edit away.
